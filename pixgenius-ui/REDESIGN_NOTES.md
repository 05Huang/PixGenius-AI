## 布局特点

1. **网格布局**: 左右两栏采用 CSS Grid，宽度为 400px + 1fr
2. **粘性定位**: 左侧面板使用 sticky 定位，滚动时固定
3. **响应式**: 
   - 1200px以下：左侧缩小到350px
   - 1024px以下：改为单栏布局
   - 640px以下：进一步缩小内边距

## 动画效果

- 按钮 hover: translateY(-2px) + 阴影
- 图片切换: fadeIn + scale 动画
- 浮动形状: 上下浮动动画(6s循环)
- 弹窗打开: fadeIn + slideUp 组合动画
- 所有交互: 0.3s 缓动过渡
